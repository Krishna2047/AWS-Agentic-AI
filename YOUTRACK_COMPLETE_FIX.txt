ERROR HANDLING
================================================================================

If 401 Unauthorized:
  - Token is invalid/expired
  - Check: backend/.env YOUTRACK_TOKEN
  - Regenerate new token if needed

If connection fails:
  - YouTrack server unreachable
  - Check: https://youtrack24.onedatasoftware.com accessible?
  - Test: curl -H "Authorization: Bearer $TOKEN" $URL/api/issues

If timeout:
  - YouTrack server slow
  - Check YouTrack status
  - Retry creation

All errors are logged clearly for debugging.
