================================================================================
CONFIGURATION OPTIONS
================================================================================

Environment Variables:
  MONITORING_ENABLED                    Boolean (default: false)
  MONITORING_CHECK_INTERVAL_MINUTES     Int (default: 15)
  MONITORING_COST_SPIKE_THRESHOLD       Float (default: 20.0)
  TEAMS_WEBHOOK_URL                     String (optional)
  SLACK_WEBHOOK_URL                     String (optional)
  DASHBOARD_URL                         String (default: http://localhost:3000)

Tuning:
  More sensitive:   Decrease threshold (10-15%), increase frequency (5 min)
  Less sensitive:   Increase threshold (30-50%), decrease frequency (60 min)
  Real-time alerts: Set interval to 5 minutes
  Cost-saving mode: Set interval to 60 minutes

QUICK START
================================================================================

1. Install dependencies:
   $ pip install -r backend/requirements.txt

2. Configure webhooks (.env file):
   TEAMS_WEBHOOK_URL=https://outlook.webhook.office.com/...
   SLACK_WEBHOOK_URL=https://hooks.slack.com/services/...

3. Start backend:
   $ cd backend && python -m uvicorn app.main:app --reload

4. Open UI:
   Go to "Continuous Monitoring" tab

5. Start monitoring:
   Click "Start Monitoring" button

6. Test webhooks:
   Click "Send Test Alert"

7. Monitor:
   Check alerts tab for detected issues
